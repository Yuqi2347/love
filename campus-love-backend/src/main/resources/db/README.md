# 数据库脚本目录说明

| 路径 | 用途 |
|------|------|
| **`migration/`** | **Flyway 版本化迁移**（`V1__…`～`V46__…`）。新环境建空库后由应用启动时自动执行，**勿**再手工跑一遍 `schema.sql`。 |
| **`schema.sql`** | 全库结构**快照**，供文档对照、diff、应急；与 `migration/` 链语义应对齐。 |
| **根目录其它 `.sql`** | 历史/运维辅助（如 `repair.sql`、`set_admin.sql`），**不**纳入 Flyway；按需手工执行并评审。 |

新增表结构请只增加 `migration/V{下一版本}__描述.sql`。
